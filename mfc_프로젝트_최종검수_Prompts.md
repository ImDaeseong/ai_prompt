## === [3단계] 잔존 이슈 최종 점검 ===

[우선순위 기준]

**[P0 - 즉시 수정 / 위험]**
- 버퍼 오버런 (strcpy, sprintf 등) / 메모리 누수 (new/delete 불일치)
- null 포인터 역참조 / 스택 오버플로우 위험

**[P1 - 조속히 수정]**
- deprecated API / 64bit 호환성 / MBCS·UNICODE 혼용 / 전역 변수 과다

**[P2 - 개선 권장]**
- 코드 중복 / 매직 넘버 / 긴 함수(>100줄) / 주석 부족

[출력 형식]

각 이슈:
```
[등급] 파일명:라인
- 문제 설명:
- 위험 이유:
- 개선 방법:
```
