# Guide: MCP Apps & Interactive UI

**MCP Apps**는 AI가 텍스트 답변뿐만 아니라 사용자와 상호작용할 수 있는 앱 인터페이스를 제공하는 개념입니다. 실제 지원 여부와 구현 방식은 사용하는 클라이언트의 공식 문서를 확인하십시오.

## 1. MCP Apps란?
기존 MCP가 데이터를 가져오는 'API' 역할이었다면, MCP Apps는 그 데이터를 시각화하고 제어할 수 있는 **'화면(UI)'**을 채팅창 안에 띄워주는 기술입니다.

## 2. 주요 활용 사례
- **금융 대시보드:** 주가 흐름을 그래프로 보여주고, 사용자가 직접 범위를 조절.
- **영상 편집기:** AI가 생성한 스토리보드를 사용자가 직접 드래그앤드롭으로 순서 변경.
- **인적 자원(HR) 도구:** 채용 후보자 리스트를 카드 형태로 보여주고 합격/불합격 버튼 클릭.

## 3. 설정 방법
- **UI 지원 서버 사용:** `npx -y @modelcontextprotocol/server-ui-dashboard` 와 같이 UI 기능을 포함한 서버를 등록해야 합니다.
- **Client 지원:** 클라이언트별 지원 범위가 다르므로 공식 문서를 확인합니다.

## 4. 사용자 팁
"이 결과를 표로 보여줘" 대신 **"이 데이터를 MCP App 대시보드로 시각화해줘"**라고 요청해 보세요. 훨씬 직관적인 제어가 가능해집니다.
