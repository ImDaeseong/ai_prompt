# Famous MCP & Skills Catalog

이 문서는 AI 에이전트 생태계에서 자주 언급되는 MCP 서버와 Skill 예시를 정리합니다. 실제 사용 전에는 공식 문서와 현재 지원 여부를 확인하십시오.

## 1. Top MCP Servers (도구 부문)

### 🔍 검색 및 리서치 (Research)
- **Brave Search:** 개인정보 보호를 중시하는 웹 검색 도구.
- **Exa / Tavily:** 일반 검색 엔진과 달리 AI가 읽기 좋은 구조화된 데이터만 골라주는 리서치 전용 도구.
- **Perplexity:** 뉴스 및 문서를 인용(Citation)과 함께 분석하는 리서치 엔진.

### ⚙️ 생산성 및 자동화 (Productivity)
- **Zapier:** Gmail, Slack, Notion 등 7,000개 이상의 앱을 AI와 연결하는 '만능 연결 고리'.
- **Notion:** 문서 관리, 위키, 프로젝트 트래킹을 AI가 직접 수행하게 함.
- **Google Calendar:** 일정 예약 및 관리를 AI 비서에게 위임.

### 💻 개발 및 데이터 (Development)
- **GitHub:** 코드 리뷰, 이슈 관리, PR 생성을 AI가 직접 수행.
- **PostgreSQL / Supabase:** 자연어로 데이터베이스를 쿼리하고 분석함.
- **Sequential Thinking:** 복잡한 문제를 단계별로 분해하고 검토하는 사고 보조 도구.
- **Context7:** 라이브러리 문서를 참조하여 코딩 문법 환각을 줄이는 문서 서버.
- **Playwright (Browser Control):** 마이크로소프트 공식 서버. AI가 직접 브라우저를 조작하여 웹 서핑 및 자동화 수행.
- **Creative Suite:** 디자인, 영상, 3D, 음악 도구와 연결하는 커넥터 범주.
- **MCP Apps:** 채팅창 내에 대시보드나 UI를 렌더링하는 인터페이스 범주.

## 2. Top Skills (지능 부문)

### ✍️ 창작 및 예술 (Creative)
- **Snowflake Writer:** (Randy Ingermanson) 단 한 문장에서 소설 전체를 설계하는 구조적 작법.
- **Master Storyteller:** 감각적 묘사와 클리셰 파괴에 특화된 서사 지침.
- **Lyrics Pro:** 음악의 리듬과 라임, 감정 선을 분석하는 작사 전문 페르소나.

### 📈 경제 및 전략 (Business)
- **Value Investor:** (Warren Buffett style) 숫자에 기반한 냉철한 기업 가치 분석.
- **Life Strategist:** AI를 활용한 수익 창출 및 생산성 최적화 로드맵 설계.
- **RTCF Expert:** 모든 업무 지시를 완벽한 구조(역할, 작업, 맥락, 형식)로 정렬.
- **Agentic Orchestrator:** 복잡한 작업을 서브 에이전트들에게 분배하고 지휘하는 중앙 통제 스킬.
- **Context Engineer:** 방대한 데이터를 AI가 효율적으로 읽도록 정보 구조를 설계하는 기술.
- **Multi-Agent Director:** CrewAI 스타일의 협업 팀 구성 및 운영 스킬.
- **Self-Correction Critic:** 답변의 오류를 스스로 찾아 교정하는 자가 비판 기술.

### 🛠️ 주요 프레임워크
- **LangGraph:** 상태 기반의 복잡한 에이전트 워크플로우 제어 표준.
- **CrewAI:** 역할 기반의 멀티 에이전트 팀 협업 프레임워크.
- **Pydantic AI:** 타입 안전성과 신뢰성을 강조한 파이썬 에이전트 SDK.
